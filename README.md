## Main Quests

### 🗡️ [SenrenTalk](https://github.com/davent10086/SenrenTalk)

面向角色扮演场景的多角色 AI 对话应用。

**Tech Stack:** React / TypeScript / Express / LangGraph / SQLite / Elasticsearch / SSE / Vitest

- 设计单聊与多角色群聊的消息状态、调用链路和前端流式展示。
- 使用 SSE 处理流式 token、消息完成、音频就绪和房间状态事件。
- 构建三层记忆结构，支持长期关系、阶段性情境和上下文延续。
- 实现群聊协调器、轮次控制、房间模式和反复读保护。
- 补充测试、架构图、单聊调用链路和群聊协调文档。

### 💬 [PomeloChat](https://github.com/davent10086/pomelo-chat)

基于 WebSocket 和 HTTP 的实时即时通讯应用。

**Tech Stack:** React / TypeScript / Vite / Ant Design / Node.js / Express / WebSocket / MySQL / Redis / JWT

- 基于 WebSocket 实现单聊和群聊的实时消息通信。
- 支持注册登录、好友管理、通讯录和群组关系管理。
- 使用 MySQL 持久化用户、好友关系和聊天消息。
- 使用 Redis 处理会话缓存、通知和实时通信相关状态。
- 支持文件传输、音视频通话信令和实验性的 AI 辅助交互。

### 📚 [Ocean](https://github.com/davent10086/ocean)

完整的图书管理与借阅后台系统。

**Tech Stack:** React / TypeScript / Ant Design / Express / Prisma / PostgreSQL / JWT / Zod / Docker / Vitest

- 实现 JWT 鉴权，以及管理员、普通成员的角色权限控制。
- 完成图书检索、分页、新增编辑、借阅、归还和库存校验。
- 使用 Prisma 管理 PostgreSQL 数据库模型、迁移和种子数据。
- 使用 Zod 完成接口参数校验，并通过统一中间件处理错误。
- 记录登录、图书、用户、借阅等关键业务操作的审计日志，并提供 Docker 本地环境。
