## Featured Projects

### [SenrenTalk](https://github.com/davent10086/SenrenTalk)

面向角色扮演场景的多角色 AI 对话应用，支持单聊、群聊、长期记忆、图片输入、语音能力与 SSE 流式输出。

**技术栈：** React、TypeScript、Express、LangGraph、SQLite、Elasticsearch、Vitest

**项目亮点：**

* 设计单聊与多角色群聊的消息状态和调用链路
* 使用 SSE 处理流式消息、轮次状态与房间结束事件
* 构建三层记忆结构，支持长期关系和上下文延续
* 实现群聊协调器、反复读保护和角色一致性约束
* 完成项目架构、调用链路、运行截图和启动文档

### [Antd-Like](https://github.com/davent10086/antd-like)

基于 React 和 TypeScript 开发的组件库，尝试从组件 API、类型设计、主题系统和测试等角度理解组件库工程。

**技术栈：** React、TypeScript、Vite、SCSS、Jest、Testing Library

**已实现组件：** Button、Input、Modal、Message、Table、Tabs、DatePicker、Dropdown、Grid、Layout 等。

### [Ocean](https://github.com/davent10086/ocean)

完整的图书管理系统，覆盖登录鉴权、角色权限、图书管理、借阅归还、用户管理、审计日志和 AI 助手。

**技术栈：** React、TypeScript、Ant Design、Express、Prisma、PostgreSQL、JWT、Zod、Docker
